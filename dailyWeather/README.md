# Daily Weather Data

This folder contains daily weather data for each Balancing Authority. There is one sub-folder for each period, historical (1983-1996), present (2016-2023), and future (2046-2053). Note that the future data is organized by Balancing Authority, while the present and historical data is organized by month-year. The present and historical data were obtained through the North American Regional Reanalysis, while the future data was calculated based on observed climate trends. All data are stored as parquet files.
