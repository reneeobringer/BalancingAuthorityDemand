# Balancing Authority Demand

This repository contains data for an analysis of electricity demand across US Balancing Authorities, given highly localized weather data from census tracts. The pre-processed demand data is in the `electricityDemand` folder, while the weather data is in the `dailyWeather` folder. Additionally, we obtained population data from the US Census Bureau, which is in the `Monthly_Tract_Population_2016_2023.csv` file. 

The data here were used to conduct retrospective and prospective analyses for shifting electrcity demand under different climate conditions. The results from this study are currently under review. 

A permanent version of this repository has been published via Zenodo: [![DOI](https://zenodo.org/badge/1263000593.svg)](https://doi.org/10.5281/zenodo.20596760).
