# Electricity Demand Data

This data was obtained from Ruggles et al. 2020 (DOI: 10.1038/s41597-020-0483-x), who used the US Energy Information Administration Form 930 (URL: https://www.eia.gov/survey/#eia-930). We leveraged the methodology published Ruggles et al. 2020 to extend the cleaned dataset to include daily demand across US Balancing Authorities through December 2023.

Here, each parquet file represents a Balancing Authority. Within the files, there are daily demand values from 2016-2023.
